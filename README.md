## Project areas I focus on

**Forecasting and predictive analytics**
I work on projects that use historical data to find patterns and produce forecasts for business-relevant scenarios. The goal is usually to make the output usable — not just technically correct.

**Dashboarding and reporting**
I build dashboards that are meant to be read by people who are not always data specialists. That means keeping things clear, structured, and focused on what actually needs to be tracked.

**Business performance analysis**
A lot of my work involves looking at operational or category-level data, identifying trends, and putting together analysis that can support decisions.

**Machine learning practice**
I use my GitHub to document what I am learning in machine learning — mostly practical exercises and structured experiments, rather than finished products. I find it useful to keep this work visible even when it is still in progress.
